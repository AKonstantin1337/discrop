This script automatically maps Discord video calls into the scene’s layout.

Discord instructions

  1. In the voice channel where the call is happening, go to the bottom-right corner and select Pop Out.
  2. In the top-right corner, ensure you’re in Grid mode (no caller appears big while the rest are small).
  3. There too, under the three dots button, ensure Show Non-Video Participants is on.

OBS capture source and scene set up

  1. Create a capture source and call it {SOURCE_NAME}; set that source to capture your Discord call. This script will only work on a source exactly named {SOURCE_NAME}.
  2. Make sure Capture Cursor is off and Client Area is on.
  3. Move it and crop it until it’s the size and position you want a single caller’s video to be in the scene— don’t pay any attention to which area of the Discord call it shows, as that’s what this script does for you.
  4. Duplicate it (always as reference) and tweak it for every other caller, across scenes, etc.
  5. Make sure the order of all {SOURCE_NAME} items in the Sources panel matches across your scenes, with the order in which you want the participants to show up (see next section).

This script set up

  1. Fill the text fields with the Discord server nicknames of each and every person/bot who might join the call throughout the stream, even if they aren’t online yet. Follow the same order you used on your {SOURCE_NAME} items in the Sources panel. Slots beyond the number of {SOURCE_NAME} items are useful for people/bots you don’t want to visually appear in the stream (e.g. Groovy).
  2. Change the levers/numbers right below each of the names to represent their status on the Discord call: left/0 meaning offline, middle/1 online but without video, and right/2 online with video.
  3. Now it’s just a matter of keeping an eye on who drops/rejoins or toggles their webcams, and pulling the levers accordingly as fast as possible!
